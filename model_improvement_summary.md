# 모델 성능 개선 결과

## 개선 방법
1. **특성 엔지니어링**: 상호작용 항 추가 (AGE×식습관, 식습관 간 상호작용)
2. **하이퍼파라미터 튜닝**: C값 조정, 트리 깊이 최적화
3. **SMOTE 오버샘플링**: 불균형 데이터 처리
4. **앙상블 방법**: 여러 모델 결합

## 1. 청소년 식습관 → 비만 예측 모델

> **레이블 정의 변경**: 2017~2023년 데이터만 사용(2015·2016 제외)하고, BMI 상위 5%(컷오프 28.39)를 비만(1)으로 지정.

| 모델 | Accuracy | Precision | Recall | F1-Score | AUC-ROC |
|------|----------|-----------|--------|----------|---------|
| **Logistic Regression (튜닝)** | 61.66% | 8.02% | **69.43%** | **14.37%** | **70.38%** |
| Random Forest (튜닝) | **78.85%** | 8.58% | 36.92% | 13.93% | 67.20% |
| Balanced Random Forest | 63.79% | 7.93% | 64.25% | 14.12% | 69.65% |
| **SMOTE + Logistic Regression** | 61.29% | 7.92% | 69.17% | 14.21% | 68.71% |
| SMOTE + Random Forest | 92.89% | **9.04%** | 5.89% | 7.13% | 68.49% |
| 앙상블 (LR + RF) | 74.22% | 8.85% | 49.03% | 14.99% | 68.84% |

**핵심 포인트**: 상위 5% 정의로 라벨이 극도로 희소해졌기 때문에 **재현율을 확보하려면 로지스틱 계열**이, **정확도를 최대화하려면 트리 계열**이 유리합니다.

## 주요 개선 사항

### ✅ 당뇨병 예측 모델
1. **Recall 향상**: 74.24% → **77.73%** (+3.49%p)
2. **Accuracy 향상**: 66.51% → **81.92%** (Random Forest 튜닝)
3. **Precision 향상**: 23.65% → **27.57%** (SMOTE 적용)

### ✅ 청소년 비만 예측 모델
1. **레이블 재정의**: BMI 상위 5%(컷오프 28.39) 기반으로 희귀 이벤트 모델링
2. **균형 맞춘 로지스틱**으로 재현율 ~0.69 확보, **앙상블**은 정확도 0.74 수준 유지

---

## 결론

1. **당뇨병 예측**: Recall이 77.73%로 향상되어 실제 당뇨병 환자를 더 많이 찾아냄
2. **비만 예측**: 앙상블 모델로 Recall과 F1-Score가 크게 개선됨
3. **특성 엔지니어링과 하이퍼파라미터 튜닝**이 모델 성능 향상에 기여
