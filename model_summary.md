# 당뇨병 위험도 예측 모델 결과 요약

## 프로젝트 개요
**청소년기 비만 및 생활습관이 성인기 당뇨병 위험에 미치는 영향 분석**

## 데이터
- **청소년 데이터**: 513,149개 행
- **성인 데이터**: 44,294개 행
- **당뇨병 유병률**: 12.62%

## 최종 모델 성능 결과 (클래스 가중치 조정 및 SMOTE 적용)

### 최고 성능 모델: Logistic Regression (Balanced)
- **Accuracy**: 66.51%
- **Precision**: 23.65%
- **Recall**: 74.24% ⭐ (높은 재현율)
- **F1-Score**: 35.88%
- **AUC-ROC**: 76.56%
- **CV AUC (5-fold)**: 76.16% (±1.13%)

### 전체 모델 비교

| 모델 | Accuracy | Precision | Recall | F1-Score | AUC-ROC |
|------|----------|-----------|--------|----------|---------|
| Logistic Regression (Balanced) | 66.51% | 23.65% | **74.24%** | **35.88%** | 76.56% |
| Logistic Regression (SMOTE) | 74.25% | 25.82% | 55.56% | 35.25% | 76.05% |
| Random Forest (Balanced) | 66.31% | 23.31% | 72.90% | 35.32% | 76.01% |
| Random Forest (SMOTE) | 72.43% | 24.83% | 58.41% | 34.85% | 75.40% |

## 특성 중요도 분석

1. **AGE (연령)**: 29.04% - 가장 중요한 예측 변수
2. **INCOME (소득)**: 17.24% - 사회경제적 지위가 당뇨병 위험에 큰 영향
3. **BREAKFAST (아침식사)**: 11.77% - 생활습관의 중요성
4. **BMI**: 8.78% - 비만도가 당뇨병 위험에 영향
5. **SEX (성별)**: 7.85% (여성) + 7.85% (남성) = 15.70%
6. **BMI_CAT_4 (비만)**: 3.48%
7. **REGION (지역)**: 2.87%

## 주요 발견사항

1. **연령이 가장 중요한 예측 변수**: 나이가 많을수록 당뇨병 위험 증가
2. **소득의 중요성**: 사회경제적 지위가 당뇨병 위험에 큰 영향을 미침
3. **생활습관의 영향**: 아침식사 빈도가 당뇨병 위험 예측에 기여 (11.77%)
4. **BMI의 영향**: 비만도가 당뇨병 위험에 영향을 미치지만, 연령과 소득보다는 낮음
5. **성별 차이**: 성별도 당뇨병 위험 예측에 기여
